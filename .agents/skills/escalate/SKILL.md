# escalate

Create a file at `.serena/memories/comms/active/<you>-to-<them>-<topic>.md` with:

```markdown
# <you> → <them>: <topic>

**Date:** <YYYY-MM-DD>
**Status:** open

## Context
- ...

## Question / Blocker
- ...

## What I've tried
- ...

## What I need
- decision / approval / information / unblock action
```

Use slug-case for `<topic>`. Confirm the file path after writing.
