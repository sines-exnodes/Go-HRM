## 6. Extracting from Figma

To extract design tokens from Figma:

1. Open Figma Desktop
2. Switch to Dev Mode (Shift+D)
3. Enable "Desktop MCP server" in inspect panel
4. Select the design frame
5. Run `/figma-design-tokens`
